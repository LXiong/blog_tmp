5.Install Java:

First download the ".tar.gz" file:
http://www.oracle.com/technetwork/java/javase/downloads/index.html
"jdk-7-linux-i586.tar.gz"

Then extract it and get the "jdk1.7.0" folder:
tar -xvzf jdk-version.tar.gz

Rename the folder:
mv jdkversion/ jdk1.7.0

Create a new folder:
sudo mkdir /usr/lib/jvm

Move the "jdk1.7.0" folder to the jvm folder:
sudo mv jdk1.7.0/ /usr/lib/jvm/

Now run:
sudo update-alternatives --install "/usr/bin/java" "java" "/usr/lib/jvm/jdk1.7.0/bin/java" 1
sudo update-alternatives --install "/usr/bin/javac" "javac" "/usr/lib/jvm/jdk1.7.0/bin/javac" 1
sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/usr/lib/jvm/jdk1.7.0/bin/javaws" 1

或者写入环境变量中: (未测试)
export JAVA_HOME=/usr/lib/jvm/jdk1.7.0
export PATH=$PATH:$JAVA_HOME/bin
export CLASSPATH=$CLASSPATH:$JAVA_HOME/lib:$JAVA_HOME/jre/lib

update-alternatives --config java

check the version:
java -version

8.Install basic server:


Install Apache:
sudo add-apt-repository ppa:gwibber-daily/ppa
sudo apt-get update
sudo apt-get install apache2

Start Apache:
sudo /etc/init.d/apache2 start
or
sudo service apache2 start

Configure Apache:
sudo gedit /etc/apache2/httpd.conf

And add this line:
ServerName localhost

Change the default folder:
sudo gedit /etc/apache2/sites-available/default

-----------------------------

Install MySQL:
sudo apt-get install mysql-server mysql-client

Check whether MySQL running:
sudo netstat -tap | grep mysql

Securing MySQL:
sudo mysql_secure_installation

Start MySQL:
sudo service mysql start
sudo service mysql restart
sudo service mysql stop

Configuration:
sudo gedit /etc/mysql/my.cnf

General Access:
mysql -u root -p

-----------------------------

Install PHP:
sudo apt-get install php5

Configure PHP:
sudo gedit /etc/php5/apache2/php.ini

Install phpMyAdmin:
sudo add-apt-repository ppa:gwibber-daily/ppa
sudo apt-get update
sudo apt-get install phpmyadmin

Then configure the phpmyadmin in apache:
sudo gedit /etc/apache2/apache2.conf

Copy this line in it:
Include /etc/phpmyadmin/apache.conf

Restart the Apache:
sudo service apache2 restart
